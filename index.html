We built Senna to be a lightweight engine that can turn any static site into a fast and responsive app without adding frameworks or rewriting tons of code.

Plug and Play

Using Senna

There are two ways to use Senna. One is by leveraging the JavaScript API and the other is by declaring HTML5 data attributes. Below you can see how to quickly get started.

1. Install

First, you can get it on npm or just use a CDN.

npm install senna

2. Include

Now add the Senna files into your head element.

<head>
    <script src="node_modules/senna/build/globals/senna.js">
    <link href="node_modules/senna/build/senna.css" rel="stylesheet">
</head>

Then, initialize it via data attributes and include the loading bar element.

<body data-senna data-senna-surface>
    <div class="senna-loading-bar"></div>
    ...	
</body>

3. Run

Finally, navigate through different pages using a slow connection to see how fast it is!

How does it work?

When the user clicks on a link, new content will be dynamically loaded without a page refresh. The application will update the URL to emulate traditional page navigation, but another full page request is never made — resulting in a much faster user experience.

1

LOAD

Your app will fetch all the resources needed on the initial page load and caches them for future use.

2

NAVIGATE

When a link is clicked, Senna prevents a full page navigation and starts fetching assets dynamically.

3

RENDER

Lastly, Senna updates the URL in the address bar and renders only the new content to the screen.
